# NHANES Data Files

## Source

**National Health and Nutrition Examination Survey (NHANES)**  
**Cycle:** 2017–March 2020 Pre-Pandemic  
**Source:** CDC Public Data Files  
**First Published:** August 2021 | **Last Revised:** NA

> **Note on cycle design:** NHANES suspended field operations in March 2020 due to COVID-19.
> Data from the incomplete 2019–2020 cycle were combined with the 2017–2018 cycle to form a
> nationally representative pre-pandemic sample. This is effectively a 2.5-cycle dataset.
> Standard 2-year survey weights do not apply — see Analytic Notes on the CDC page before
> running population-level inference.

## P_RHQ Module: Reproductive Health Questionnaire

**5,314 participants × 32 variables** | Females only | Ages 12–150 years (varies by variable)

### Key Variables by Domain

#### Menstrual History

| Variable | Description | Notes |
| --- | --- | --- |
| RHQ010 | Age at first menstrual period | 0 = not yet started; 20 = 20+ years; range 6–19 |
| RHD018 | Estimated age at menarche | **In months, not years** — divide by 12 before use; range 103–191 months |
| RHQ020 | Age range at first menstrual period | Ordinal fallback: 1=<10, 2=10–12, 3=13–15, 4=16+ |
| RHQ031 | Regular periods in past 12 months | 1=Yes, 2=No |
| RHD043 | Reason for no periods in past 12 months | 1=Pregnancy, 2=Breastfeeding, 3=Hysterectomy, 7=Menopause, 9=Other |
| RHQ060 | Age at last menstrual period | 19=19 or under; range 20–59; 60=60+ |
| RHQ070 | Age range at last menstrual period | Ordinal fallback: 1=<30, 2=30–34, 3=35–39, 4=40–44, 5=45–49, 6=50–54, 7=55+ |

#### Infertility & Pelvic History

| Variable | Description | Notes |
| --- | --- | --- |
| RHQ074 | Tried ≥1 year to become pregnant without success | Primary infertility / PCOS proxy |
| RHQ076 | Seen a doctor because unable to become pregnant | Secondary infertility marker |
| RHQ078 | Ever treated for pelvic infection / PID | Fertility and surgical risk context |

#### Pregnancy History

| Variable | Description | Notes |
| --- | --- | --- |
| RHQ131 | Ever been pregnant | 1=Yes, 2=No |
| RHD143 | Currently pregnant | 1=Yes, 2=No; ages 20–44 only |
| RHQ160 | Total number of pregnancies (gravidity) | Range 1–10; 11 = 11 or more |
| RHD167 | Total number of deliveries (parity) | Range 0–4; 5 = 5 or more |
| RHQ171 | Deliveries resulting in live birth | Range 0–4; 5 = 5 or more |
| RHQ197 | Months since last baby | Range 2–24; 1 = 1 month or less |
| RHQ200 | Currently breastfeeding | 1=Yes, 2=No |

#### Adverse Pregnancy Outcomes

| Variable | Description | Clinical Significance |
| --- | --- | --- |
| RHQ162 | Gestational diabetes during pregnancy | 7× ↑ lifetime T2DM risk; 2× CVD risk. Code 3 = borderline → **treated as Yes** in this project |
| RHQ172 | Any baby ≥9 lbs at birth (macrosomia) | Marker of maternal insulin resistance even without GDM diagnosis |

> ⚠️ **Variables not present in this release:** RHQ163 (hypertension in pregnancy),
> RHQ166 (pre-eclampsia), and RHQ169 (excess gestational weight gain) do not exist in the
> P_RHQ 2017–2020 file. These were available in earlier NHANES cycles. HDP and excess GWG
> features cannot be derived from this module alone.

#### Birth Timing

| Variable | Description | Notes |
| --- | --- | --- |
| RHD180 | Age at first live birth | 17 = 17 or younger; range 18–43; 45 = 45 or older |
| RHD190 | Age at last live birth | 17 = 17 or younger; range 18–44; 45 = 45 or older |

#### Surgical History

| Variable | Description | Clinical Significance |
| --- | --- | --- |
| RHD280 | Had a hysterectomy | Uterus removal; ovarian status requires RHQ305 |
| RHQ305 | Both ovaries removed | Surgical menopause; accelerated CVD risk |
| RHQ332 | Age when both ovaries removed | 19 = 19 or under; range 22–59; 60 = 60 or older |

#### Hormone Therapy

| Variable | Description | Notes |
| --- | --- | --- |
| RHQ540 | Ever used female hormones (any form) | Screening question; gate for RHQ542A–D |
| RHQ542A | Hormone therapy: pills | Multi-select; 10 = Yes |
| RHQ542B | Hormone therapy: patches | Multi-select; 11 = Yes |
| RHQ542C | Hormone therapy: cream/suppository/injection | Multi-select; 12 = Yes |

## Project Analytic Decisions

These decisions are documented here and in `01_reproductive_feature_engineering.ipynb`:

| Decision | Rationale |
| --- | --- |
| Borderline GDM (RHQ162 code 3) → coded as Yes | ADA evidence supports similar downstream T2DM and CVD risk as overt GDM |
| RHD018 divided by 12 before use | Variable is in months; all other age variables are in years |
| RHQ160 used for gravidity, RHD167 for parity | Correct obstetric distinction: gravidity = pregnancies, parity = deliveries |
| RHQ074 used as infertility/PCOS proxy | No direct PCOS diagnosis in NHANES; trying ≥1 year without conceiving is the closest available marker |
| Post-hysterectomy without BSO treated as separate menopausal category | Ovarian status is indeterminate without bilateral oophorectomy confirmation |
| RHQ070 midpoint imputation for Age_Menopause | Where RHQ060 is missing, ordinal range midpoint used as approximation |

## Missing Data Codes

Standard NHANES coding applied uniformly across all variables:

| Code | Meaning | Action |
| --- | --- | --- |
| `.` (blank) | Not asked / skip logic | Keep as NaN — structurally missing, not random |
| `7`, `77`, `777` | Refused | → NaN |
| `9`, `99`, `999` | Don't know | → NaN |

> **Important:** Structural missingness (blank / not asked) must not be imputed. A woman
> who was never asked about gestational diabetes because she was never pregnant is
> fundamentally different from one who refused to answer.

## How to Obtain Data

### Option 1: Automated Download (Recommended)

```bash
python scripts/download_nhanes_data.py
```

This script downloads all required modules, skips files already present, and verifies
file sizes. Total download: ~9 MB.

### Option 2: Verify Existing Files

```bash
python scripts/download_nhanes_data.py --verify
```

### Option 3: Manual Download

1. Click the download links in the Files Required table above
2. Save `.XPT` files to `data/raw/`
3. Filenames are case-sensitive — ensure they match exactly

## Data Exclusion from Repository

Raw `.XPT` files are excluded from version control (see `.gitignore`) because they are
publicly available from CDC and reproducible via the download script. Processed outputs
in `data/processed/` are also excluded — run the notebooks to regenerate them.

## File Formats

NHANES distributes data as **SAS XPORT transport files (.XPT)**. Read in Python with:

```python
import pandas as pd
df = pd.read_sas('data/raw/P_RHQ.XPT', format='xport', encoding='utf-8')
```
